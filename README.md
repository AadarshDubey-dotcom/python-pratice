# Python Practice

This repository contains simple Python practice examples.

## Project Structure

- `Basic/Variable.py` - Demonstrates basic Python concepts including:
  - What Python is
  - What a variable is
  - How to declare variables in Python
  - Example variables for `name`, `age`, and `height`

## How to run

1. Open the folder in a Python-capable editor or terminal.
2. Run the example file:

```bash
python Basic/Variable.py
```

## Notes

- `Variable.py` uses comments to explain Python and variable basics.
- This repository is a starting point for learning Python syntax and variable assignment.
